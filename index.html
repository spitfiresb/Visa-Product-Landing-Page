The trust layer for agentic commerce

AI agents are beginning to make purchases on behalf of users. The Agent Identity Token (AIT) is standardized data that Visa requires for agent transactions to qualify for protection programs.

Open protocols generate AITs. Visa validates them—ensuring every agent transaction is authorized, secure, and protected.
